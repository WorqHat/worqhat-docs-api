# System Info

Returns detailed information about the API system, including version, environment, and available features.

```endpoint
GET /system/info
```

## Response

<ResponseField name="version" type="string">
  The current version of the API
</ResponseField>

<ResponseField name="environment" type="string">
  The environment the API is running in (e.g., "production", "staging")
</ResponseField>

<ResponseField name="features" type="array">
  List of available features and their status
</ResponseField>

<ResponseField name="limits" type="object">
  Current rate limits and quotas for the authenticated user
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.worqhat.com/system/info" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript Node.js
import Worqhat from 'worqhat';

const client = new Worqhat({
  apiKey: process.env.WORQHAT_API_KEY,
});

async function getSystemInfo() {
  try {
    const response = await client.system.info();
    console.log(`API Version: ${response.version}`);
    console.log(`Environment: ${response.environment}`);
    console.log('Available Features:', response.features);
    return response;
  } catch (error) {
    console.error('Error getting system info:', error);
  }
}

getSystemInfo();
```

```python Python
from worqhat import Worqhat

# Initialize the client
client = Worqhat(
    api_key="YOUR_API_KEY",  # Better to use environment variables
)

# Get system info
response = client.system.info()
print(f"API Version: {response.version}")
print(f"Environment: {response.environment}")
print(f"Available Features: {response.features}")
```

```go Go
package main

import (
  "context"
  "fmt"

  "github.com/WorqHat/worqhat-go-sdk"
  "github.com/WorqHat/worqhat-go-sdk/option"
)

func main() {
  client := worqhat.NewClient(
    option.WithAPIKey("YOUR_API_KEY"),  // Better to use environment variables
  )
  
  response, err := client.System.Info(context.TODO())
  if err != nil {
    panic(err.Error())
  }
  
  fmt.Printf("API Version: %s\n", response.Version)
  fmt.Printf("Environment: %s\n", response.Environment)
  fmt.Println("Available Features:", response.Features)
}
```
